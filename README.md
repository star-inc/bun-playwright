# bun-playwright

Docker image for running Playwright tests on Bun.

## Overview

- Bun-based runtime with Playwright pre-installed.
- Includes browser dependencies and a ready-to-test environment.
- Built for fast containerized browser automation testing.

## Quickstart

```bash
# Pull the latest image from GitHub Container Registry
docker pull ghcr.io/star-inc/bun-playwright:main
```

## Supported platform

- Bun
- Playwright
